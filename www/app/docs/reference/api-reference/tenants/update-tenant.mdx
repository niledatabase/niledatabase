# Update a Tenant

    <div className="grid grid-cols-2 gap-4">
    You can rename tenants with a PUT request.
    <CodeGroups
      title={"Update Tenant Request"}
      method={"PUT"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/tenants/:tenantId"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces/<string>/databases/<string>/tenants/<string>' \
      --header 'Authorization: Bearer <string>' \
      --data '{
        "name": "<string>"
      }'
      ```
    </CodeGroups>
    The response includes the generated tenant id and the name you provided.
    <CodeGroups
      title={"Update Tenant Response"}
