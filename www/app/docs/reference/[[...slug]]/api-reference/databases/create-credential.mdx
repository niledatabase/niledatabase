# Create Database Credential

You'll need credentials in order to connect to your database from your database client or application.

<Row>
  <Col>
    Create a new credential with an empty POST request.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Credential Request"}
      method={"POST"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/credentials"}
    >
      ```bash
      curl --location --request POST 'https://api.thenile.dev/workspaces/<string>/databases/<string>/credentials' \
      --header 'Authorization: Bearer <string>'
      ```
    </CodeGroups>
  </Col>
</Row>
<Row>
  <Col>
    We'll generate a password for you and return it in the response. This is the only time you'll see it! Credentials are stored as salted and hashed values in the `auth.credentials` table of your database. The response also includes the unique id for the credential and the database information.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Credential Response"}
      response={"201"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/credentials"}
    >
      ```json
      {
        "id": "018b61d5-9b55-7f8e-a4ef-3e87cfb99c74",
        "database": {
          "id": "018b61c8-d311-7178-9219-0ecb5594c65f",
          "name": "database_one",
          "workspace": {
            "name": "My Workspace",
            "slug": "my-workspace"
          }
        },
        "password": "e52079bf-5252-4427-8c0d-e79599fbc97b"
      }
      ```
    </CodeGroups>
  </Col>
</Row>
