# Create Database

    <div className="grid grid-cols-2 gap-4">
    Create a new database by providing a database name. Names must be unique within a workspace, and they must follow Postgres naming rules. The name must start with a letter or an underscore and can contain only letters, numbers, and underscores. The name cannot be longer than 63 characters.
    <CodeGroups
      title={"Create Database Request"}
      method={"POST"}
      pathname={"/workspaces/:workspaceSlug/databases"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces/<string>/databases' \
      --header 'Authorization: Bearer <string>' \
      --data '{
        "databaseName": "<string>"
      }'
      ```
    </CodeGroups>
    The response contains the database id, name, and workspace information.
    <CodeGroups
      title={"Create Database Response"}
      response={"201"}
      pathname={"/workspaces/:workspaceSlug/databases"}
    >
      ```json
      {
        "id": "018b61bc-cf2c-7392-92a6-f1a811cf4419",
        "name": "database_one",
        "workspace": {
          "name": "My Workspace",
          "slug": "my-workspace"
        }
      }
      ```
    </CodeGroups>

</div>
