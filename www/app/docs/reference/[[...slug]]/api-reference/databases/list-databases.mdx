# List Databases

    <div className="grid grid-cols-2 gap-4">
    List the databases in the workspace.
    <CodeGroups
      title={"List Database Request"}
      method={"GET"}
      pathname={"/workspaces/:workspaceSlug/databases"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces/<string>/databases' \
      --header 'Authorization: Bearer <string>'
      ```
    </CodeGroups>
