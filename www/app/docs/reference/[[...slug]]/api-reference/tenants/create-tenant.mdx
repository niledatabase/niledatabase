# Create a Tenant

<Row>
  <Col>
    Create a new tenant by providing a tenant name. Tenant names must be unique within a database.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Tenant Request"}
      method={"POST"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/tenants"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces/<string>/databases/<string>/tenants' \
      --header 'Authorization: Bearer <string>' \
      --data '{
        "name": "<string>"
      }'
      ```
    </CodeGroups>
  </Col>
</Row>
<Row>
  <Col>
    The response includes the generated tenant id and the name you provided.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Tenant Response"}
      response={"201"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/tenants"}
    >
      ```json
      {
        "id": "018b61ec-af9e-77e5-b27f-77239cd4a0bc",
        "name": "Tenant Zero"
      }
      ```
    </CodeGroups>
  </Col>
</Row>
