# Get a Tenant

<div className="grid grid-cols-2 gap-4">

    Get a specific tenant by its id.
    <CodeGroups
      title={"Get Tenant Request"}
      method={"GET"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/tenants/:tenantId"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces/<string>/databases/<string>/tenants/<string>' \
      --header 'Authorization: Bearer <string>'
        ```
    </CodeGroups>

    The response includes the id and name of the tenant.
    <CodeGroups
      title={"Get Tenant Response"}
      response={"200"}
      pathname={"/workspaces/:workspaceSlug/databases/:databaseName/tenants/:tenantId"}
    >
      ```json
      {
        "id": "018b61ec-af9e-77e5-b27f-77239cd4a0bc",
        "name": "Tenant Zero"
      }
      ```
    </CodeGroups>

</div>
