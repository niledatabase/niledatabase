# Create a Workspace

<Row>
  <Col>
    Create a new workspace by providing a workspace name. Workspace names must be unique across Nile. Your company or application name is a good choice for a workspace name.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Workspace Request"}
      method={"POST"}
      pathname={"/workspaces"}
    >
      ```bash
      curl 'https://api.thenile.dev/workspaces' \
      --header 'Authorization: Bearer <string>' \
      --data '{
        "name": "<string>"
      }'
      ```
    </CodeGroups>
  </Col>
</Row>
<Row>
  <Col>
    The response will include your workspace name and slug. The name is the human-readable name you provided. The slug is a URL-safe version of the name that you'll use in API paths.
  </Col>
  <Col>
    <CodeGroups
      title={"Create Workspace Response"}
      method={"201"}
      pathname={"/workspaces"}
    >
      ```json
      {
        "name": "My Workspace",
        "slug": "my-workspace"
      }
      ```
    </CodeGroups>
  </Col>
</Row>
